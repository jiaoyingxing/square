# Square 0.1.0

[中文说明](RELEASE_NOTES-zh.md)

Square 0.1.0 is the first public build and focuses on the core daily loop.

## What changed in this version

- Create and edit projects from the plugin settings.
- Choose built-in templates without keeping template files in your vault.
- Complete check-ins from one quick entry modal with record text, date, count, fixed fields, and attachments.
- Review week, month, and ranking views in the main view or sidebar.
- Plan which projects should appear on future dates from the month view.
- Keep project records in plain Markdown notes instead of switching to a separate system.

## Install

Download `main.js`, `manifest.json`, and `styles.css` from this repository, put them into `.obsidian/plugins/square/`, then restart Obsidian or reload plugins.

## Who it is for

For Obsidian users who want daily check-ins, recurring projects, and target tracking to stay inside Markdown notes.
