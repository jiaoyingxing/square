# Square 小方块

Square 小方块是一层放在 Obsidian 里的可视化打卡入口。

你每天点的是小方块，留下来的仍然是 Markdown 记录。

如果你已经把目标、习惯、日常项目和复盘内容放在 Obsidian 里，但打卡还要跳到另一个 App，Square 解决的就是这件事：把“今天要做什么、刚才做了什么、最近坚持得怎么样”放回你的笔记库。

它不是提醒软件，不是习惯社区，也不是一套新的数据库。它更像是一个轻量操作层：项目变成小方块，记录写回项目笔记，需要回看时再打开图表。

## 它到底是什么

Square 把 Obsidian 里的打卡项目拆成几层：

- 小方块主界面：每天打开后，先看到今天要处理的项目。
- 快速记录弹窗：点开项目后，记录文字、日期、次数、固定字段和附件。
- Markdown 项目笔记：每个项目都有自己的笔记，历史记录仍然可读、可迁移、可手动检查。
- 内置模板：帮你从单次打卡、多次计数、目标追踪、热力图、趋势记录等常见场景开始。
- 总览图表：需要回看时，再用周视图、月视图和排行视图看近期状态。

所以它的本质不是“再做一个打卡 App”，而是给 Obsidian 增加一个更直观、更顺手的打卡层。

## 日常怎么用

1. 在插件设置里创建一个项目，比如阅读、运动、喝水、写作、背单词。
2. 选择一个适合的模板，比如单次完成、多次计数、目标进度或趋势记录。
3. 回到 Square 主界面，项目会显示成一个小方块。
4. 到了该记录的时候，点开小方块，写下记录、日期、次数或固定字段。
5. 记录会保存到对应的 Markdown 项目笔记里。
6. 想回看时，打开周视图、月视图或排行视图。
7. 如果某些未来日期只想显示一部分项目，可以在月视图里提前安排。

日常路径可以很短：打开 Square，看方块，记录，结束。

## 和一般打卡软件有什么不一样

一般打卡软件通常围绕提醒、连续天数、社交监督、积分或云端账号设计。Square 的重点不在这里。

Square 更关心四件事：

- 数据不离开你的笔记库：记录保存在 Obsidian 的 Markdown 文件里，而不是被关进另一个 App。
- 操作要足够轻：主界面先给你今天要点的方块，不先让你面对复杂后台。
- 场景不只是一枚勾选框：有些项目只需要完成，有些需要次数，有些需要目标，有些适合看趋势。
- 回看不打扰记录：图表负责复盘，主界面负责今天的行动，两者不互相抢位置。

如果你需要的是强提醒、社交打卡、跨平台云同步或完整习惯社区，Square 不是那类产品。

如果你想把打卡留在 Obsidian 里，并且希望它比普通任务列表更直观，Square 更接近这个需求。

## 现在可以做什么

- 创建和编辑 Square 项目。
- 使用内置模板开始常见打卡场景。
- 在小方块主界面里快速记录。
- 记录文字、日期、次数、固定字段和附件。
- 把每个项目保存为 Markdown 笔记。
- 查看周视图、月视图和排行视图。
- 从月视图安排未来日期要显示的项目。
- 归档暂停或结束的项目。
- 在 Obsidian 桌面端和移动端使用。

## 适合谁

Square 更适合这些用户：

- 已经把日常记录放在 Obsidian 里。
- 想要一个比任务列表更直观的打卡界面。
- 不想为了打卡再维护一套独立数据库。
- 希望记录能长期保存在 Markdown 里。
- 需要同时记录“完成了没有”和“做了多少、写了什么”。

## 授权与版本

Square 小方块是专有软件。

未授权状态适合用少量项目试用核心流程。授权后可以解锁更完整的项目使用。

授权密钥通过 Obsidian 内置密钥管理保存，并在本地完成签名验证。正常本地使用不需要额外的服务器账号系统。

如需获取授权密钥，可以在小红书搜索 `焦应行`。

## 隐私与存储

- 项目记录会保存为你 Obsidian 笔记库里的 Markdown 笔记。
- 插件设置保存在本地 Obsidian 插件数据里。
- 授权密钥通过 Obsidian SecretStorage 保存。
- Square 目前不包含客户端遥测。
- 附件会根据插件流程和你的笔记库结构保存到本地库中。

不要公开你的 `.obsidian/plugins/square/data.json`。它可能包含本地设置、授权状态和历史配置数据。

## 安装

Square 通过 GitHub Releases 分发。

由于当前 Obsidian 官方插件商店规则要求上架插件开源，Square 在这个版本不会上架官方插件商店。

### 推荐方式：用 BRAT 安装

1. 先从 Obsidian 第三方插件里安装 `BRAT`。
2. 打开 BRAT 设置，选择 `Add Beta plugin`。
3. 填入这个仓库地址：

```text
https://github.com/jiaoyingxing/square
```

4. 让 BRAT 安装最新 release。
5. 在 Obsidian 第三方插件设置中启用 `Square`。

推荐用 BRAT，是因为它后续可以帮你从 GitHub Releases 更新，不用每次手动替换文件。

### 手动安装

1. 从最新 GitHub Release 下载 `main.js`、`manifest.json` 和 `styles.css`。
2. 在你的笔记库里创建这个目录：

```text
.obsidian/plugins/square/
```

3. 把下载的三个文件放进去。
4. 重启 Obsidian，或重新加载第三方插件。
5. 在 Obsidian 设置中启用 `Square`。

不要把 GitHub 自动生成的源码压缩包当成安装包。请使用 Release assets 里的安装文件。
