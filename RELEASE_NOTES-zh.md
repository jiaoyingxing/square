# Square 0.1.0

[English](RELEASE_NOTES.md)

Square 0.1.0 是第一个公开安装版本，先把最核心的日常闭环跑通。

## 这版主要带来了什么

- 你可以在插件设置里创建和编辑项目。
- 你可以直接选内置模板，不用再维护模板文件。
- 你可以用一个快速记录弹窗完成记录、日期、次数、固定字段和附件。
- 你可以在主界面或侧栏查看周视图、月视图和排行视图。
- 你可以从月视图提前安排未来日期要显示哪些项目。
- 你仍然是在普通 Markdown 笔记里保存项目记录，不需要换到另一套系统。

## 安装说明

推荐用 BRAT 安装和更新：

1. 从 Obsidian 第三方插件里安装 `BRAT`。
2. 在 BRAT 里添加 beta 插件：`https://github.com/jiaoyingxing/square`。
3. 让 BRAT 安装最新 release，然后启用 `Square`。

手动安装也可以：从本 release 下载 `main.js`、`manifest.json` 和 `styles.css`，放入你的 Obsidian 笔记库目录 `.obsidian/plugins/square/`，然后重启 Obsidian 或重新加载插件。

Square 小方块是专有软件。由于 Obsidian 官方插件商店的新规则不支持非开源插件上架，所以暂不走官方插件商店分发。

## 适合谁

适合想把日常打卡、重复项目和目标记录留在 Markdown 笔记里的 Obsidian 用户。
