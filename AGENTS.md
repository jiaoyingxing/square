# Square Public 仓入口

## 先读

1. 需要公开介绍时读 `README.md`
2. 需要中文介绍时读 `README-zh.md`
3. 需要版本说明时读 `RELEASE_NOTES.md` 或 `RELEASE_NOTES-zh.md`
4. 需要运行事实时读 `manifest.json` 和 `versions.json`

## 仓库边界

- 这里是 Square 的公开发布仓，不是源码仓。
- 这里只放安装构筑物和用户向发布材料。
- 不在这里查源码、开发日志、核心文档或内部决策。

## 维护口径

- 公开说明继续写在 README，不写进入口层。
- 私有源码事实回到 `E:\Code\square`。
